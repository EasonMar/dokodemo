# dokodemo door（文件任意门）

## 项目概述

这是一个 uTools 插件，名为 "dokodemo door"，中文描述为 "文件任意门"。该插件提供了一个直观的文件管理界面，允许用户通过拖拽、选择和操作文件，实现文件的快速复制、移动和删除等功能。

## 核心功能

1. **文件拖拽功能**：
   - 支持将文件拖拽到 INPUT 或 OUTPUT 区域
   - 自动解析拖拽的文件和文件夹结构

2. **文件浏览功能**：
   - 可展开/折叠目录结构
   - 显示文件和文件夹的名称

3. **文件操作功能**：
   - 复制文件：从 INPUT 区域复制到 OUTPUT 区域
   - 删除文件：删除选中的文件或文件夹
   - 移动项目：在 INPUT 和 OUTPUT 区域之间移动项目

4. **操作管理功能**：
   - 清空区域内容
   - 撤销选择
   - 移出项目

## 技术实现

### 项目结构

| 文件        | 功能                                 |
| ----------- | ------------------------------------ |
| plugin.json | 插件配置文件，定义插件基本信息和命令 |
| index.html  | 插件主界面结构                       |
| index.js    | 插件主要逻辑实现                     |
| preload.js  | 提供底层文件操作功能                 |
| style.css   | 样式文件                             |
| logo.png    | 插件图标                             |

### 核心技术点

1. **文件系统操作**：
   - 使用 Node.js 的 fs 模块实现文件读写、复制和删除操作
   - 通过 preload.js 暴露文件操作函数到前端

2. **界面交互**：
   - 使用 HTML5 的拖放 API 实现文件拖拽功能
   - 使用 jQuery 简化 DOM 操作和事件绑定
   - 动态生成文件和文件夹的树形结构

3. **uTools 集成**：
   - 使用 uTools API 实现插件的初始化和与 uTools 的交互
   - 通过命令 "dd"、"fme" 或 "dokodemo door" 启动插件

4. **数据管理**：
   - 使用数组存储 INPUT 和 OUTPUT 区域的文件信息
   - 维护选中文件的状态

## 工作流程

1. **插件启动**：
   - 用户通过命令 "dd"、"fme" 或 "dokodemo door" 启动插件
   - 插件自动读取当前文件夹路径并添加到 OUTPUT 区域

2. **文件添加**：
   - 用户可以通过拖拽方式添加文件到 INPUT 或 OUTPUT 区域
   - 插件会解析文件结构并显示在界面上

3. **文件操作**：
   - 用户可以展开/折叠目录查看文件结构
   - 用户可以选择文件或文件夹
   - 用户可以执行复制、删除等操作

4. **操作执行**：
   - 点击 "复制到Output" 按钮将 INPUT 选中的文件复制到 OUTPUT 选中的目录
   - 点击 "删除所选" 按钮删除选中的文件或文件夹
   - 点击 "添加到Input" 按钮将 OUTPUT 选中的项目移动到 INPUT 区域

## 代码分析

### 核心功能实现

1. **文件系统操作**（preload.js）：
   - `readDir`：读取目录内容
   - `isDir`：判断路径是否为目录
   - `copyTo`：复制文件
   - `deleteFile`：删除文件或文件夹

2. **界面生成**（index.js）：
   - `DomCreateing`：根据数据生成文件列表界面
   - `generateList`：递归生成文件和文件夹的树形结构
   - `EventBinding`：绑定界面元素的事件处理函数

3. **事件处理**（index.js）：
   - 拖拽事件：处理文件拖拽到界面的操作
   - 点击事件：处理文件选择、目录展开/折叠等操作
   - 按钮事件：处理清空、撤销、复制、删除等操作

### 技术亮点

1. **模块化设计**：将文件系统操作和界面逻辑分离，提高代码可维护性

2. **响应式界面**：使用 HTML5 和 CSS 构建直观的用户界面

3. **跨平台兼容**：通过 `utools.isWindows()` 检测操作系统，使用不同的路径分隔符

4. **用户体验优化**：
   - 支持拖拽操作，提高文件添加的便捷性
   - 实时反馈操作结果
   - 提供撤销和清空功能，方便用户纠正操作

## 总结

"dokodemo door" 是一个功能实用的 uTools 插件，通过直观的界面和便捷的操作，帮助用户更高效地管理文件。它实现了文件的拖拽、浏览、复制和删除等核心功能，为用户提供了一种快速操作文件的方式。

该插件的代码结构清晰，逻辑分明，使用了现代前端技术和 Node.js 文件系统 API，展示了如何开发一个功能完整的 uTools 插件。通过进一步的优化和功能扩展，这个插件可以成为用户日常文件管理的得力助手。
