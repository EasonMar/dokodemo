# 文件重命名功能实现

我已经为 "dokodemo door" 插件添加了文件重命名功能，特别是支持使用正则表达式进行批量重命名。以下是实现的详细内容：

## 功能实现

### 1. 底层文件操作

- 在 `preload.js` 中添加了 `renameFile` 函数，用于执行文件重命名操作
- 修复了 `copyTo` 和 `deleteFile` 函数，添加了缺失的 callback 参数

### 2. 界面元素

- 在 `index.html` 中添加了 "重命名所选" 按钮
- 添加了重命名对话框，包含两种重命名模式：
  - 直接重命名：输入新名称，自动保留文件扩展名
  - 正则替换：使用正则表达式匹配和替换文件名

### 3. 样式设计

- 在 `style.css` 中添加了重命名对话框的样式
- 实现了平滑的动画效果和响应式布局

### 4. 核心逻辑

- 在 `index.js` 中实现了重命名功能的完整逻辑：
  - 显示/隐藏重命名对话框
  - 处理重命名模式切换
  - 实时生成重命名预览
  - 执行重命名操作
  - 处理错误和成功反馈
  - 更新内存中的文件路径信息

## 使用方法

1. **选择文件**：在 INPUT 或 OUTPUT 区域选择一个或多个文件或文件夹
2. **打开重命名对话框**：点击 "重命名所选" 按钮
3. **选择重命名模式**：
   - **直接重命名**：输入新名称，系统会自动保留文件扩展名
   - **正则替换**：输入正则表达式和替换内容
4. **预览结果**：对话框会实时显示重命名前后的文件名对比
5. **执行重命名**：点击 "确认" 按钮执行重命名操作
6. **查看结果**：操作完成后会显示成功或失败的提示信息

## 正则表达式使用示例

1. **替换文件名中的特定字符**：
   - 正则表达式：`old`
   - 替换为：`new`
   - 效果：将文件名中的 "old" 替换为 "new"

2. **添加前缀**：
   - 正则表达式：`^`
   - 替换为：`prefix_`
   - 效果：在文件名开头添加 "prefix\_"

3. **添加后缀**：
   - 正则表达式：`$`
   - 替换为：`_suffix`
   - 效果：在文件名末尾添加 "\_suffix"

4. **批量编号**：
   - 正则表达式：`^(.*)$`
   - 替换为：`$1_001`
   - 效果：在文件名末尾添加 "\_001"

5. **提取特定部分**：
   - 正则表达式：`.*?(\d+).*`
   - 替换为：`file_$1`
   - 效果：提取文件名中的数字部分并构建新文件名

## 技术亮点

1. **用户友好的界面**：直观的对话框设计，实时预览功能
2. **强大的正则支持**：支持复杂的文件名模式匹配和替换
3. **批量操作**：可以同时重命名多个文件
4. **错误处理**：完善的错误捕获和提示机制
5. **实时反馈**：操作过程中显示加载状态，操作完成后显示结果

这个功能的实现大大增强了插件的实用性，特别是在需要批量处理文件名的场景中，使用正则表达式可以实现复杂的重命名需求。
